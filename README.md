# Mr. Maggot

Mr. Maggot is an application that functions to monitor and control maggot cultivation with the help of a microcontroller.

## Application build with

- Kotlin.
- Firebase.
- Clean architecture.

## Installation

Clone project
```
https://github.com/dwitio123/Maggot.git
```
